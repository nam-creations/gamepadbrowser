# gamepadbrowser
Gamepad Browser  - Documentation &amp; Support

Instantly play PC web games with a virtual pad! Perfect for Itch.io and PICO-8.

[Enjoy PC browser games on your smartphone]
This is a dedicated gaming web browser developed for playing web games that require keyboard controls on your smartphone.
"I want to play free PC games on my smartphone, but I don't have a keyboard..."
This solves that problem.
Simply launch the app and open a URL, or select this app via browser sharing, and a virtual gamepad will appear on-screen.
Simulate inputs like WASD, arrow keys, and spacebar with touch controls.

[Key Features]
Equipped with a virtual controller: Displays a D-pad and A/B buttons on-screen. Intuitively control PC games.
Free key mapping: Freely assign arrow keys, WASD, Z/X/Space, and more to suit your game.
General-purpose browser functionality: Supports URL input and Google search. Access web games from around the world, not just specific sites.

[Recommended for]
Indie game fans: Those who want to easily play popular horror and action games on their smartphone.
Game commentators: People who want to use just their smartphone to find ideas for web games or record gameplay videos.
Game developers: Creators who want their PC web games to be playable by smartphone users (even if touch controls are not implemented, this browser will work).

[Examples of supported sites]
This app supports common game sites that run on HTML5 and WebGL (such as Itch.io), but we cannot guarantee that all games will work.

[For developers]
This app sends standard keyboard events to web pages.
It can also be used as a debugging tool to adapt your games for mobile.
